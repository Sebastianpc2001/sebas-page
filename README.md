# sebas-page

Minimal personal website inspired by https://dhrumil.ca/.

## Goal
Create a clean, static personal site in plain HTML/CSS/JS with a similar minimalist feel:
- short intro
- current focus
- writing/projects links
- location/time footer detail
- fast, simple, easy to host on GitHub Pages

## Project plan

### Phase 1
- Create a static one-page site in HTML/CSS/JS
- Replicate the minimalist structure and spacing, without copying content
- Make it easy to customize

### Phase 2
- Add secondary pages for notes/projects/books/photos if wanted
- Improve typography and responsive spacing
- Add Open Graph metadata and favicon

### Phase 3
- Deploy to GitHub Pages
- Connect custom domain later if desired

## Stack
- HTML
- CSS
- Vanilla JavaScript

## Local preview
Open `index.html` directly or serve with any static server.
